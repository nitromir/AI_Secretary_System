# Settings (Настройки)

Профиль, безопасность, общие настройки системы, тема, язык и управление данными.

**Доступ:** `user` и выше | **Путь:** `/settings`

## Профиль

### Информация о пользователе

| Поле | Описание |
|------|----------|
| **Имя пользователя** | Логин (только чтение) |
| **Роль** | admin / user / web / guest (только чтение) |
| **Дата создания** | Когда создан аккаунт |

### Смена пароля

1. Введите текущий пароль
2. Введите новый пароль (мин. 4 символа)
3. Подтвердите новый пароль
4. Нажмите "Изменить пароль"

**API:** `POST /admin/auth/change-password`

## Интерфейс

### Язык

| Язык | Код |
|------|-----|
| Русский | `ru` |
| English | `en` |

Переключение мгновенное, без перезагрузки.

### Тема

| Тема | Описание |
|------|----------|
| **Light** | Светлая тема |
| **Dark** | Тёмная тема |
| **Night Eyes** | Тёплая тёмная (для глаз) |

Тема сохраняется в localStorage.

### Быстрый экспорт

| Тип | Описание |
|-----|----------|
| **FAQ** | Экспорт только FAQ записей |
| **TTS пресеты** | Экспорт TTS пресетов |
| **Полный** | Все настройки, персоны, FAQ, пресеты, боты, виджеты |

### Быстрый импорт

1. Нажмите "Импорт"
2. Выберите JSON файл
3. Подтвердите

⚠️ **Внимание:** Импорт перезаписывает существующие данные!
